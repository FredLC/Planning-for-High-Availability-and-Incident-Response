# API Service


| Category     | SLI                                                    | SLO                                                                                                         |
| ------------ | ------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------- |
| Availability | The percentage of successful requests over the last 5m | 99%                                                                                                         |
| Latency      |  The 90th percentile of request response times.                                                      | 90% of requests below 100ms                                                                                 |
| Error Budget | The remaining error budget                                                       | Error budget is defined at 20%. This means that 20% of the requests can fail and still be within the budget |
| Throughput   | The number of successful requests per second                                                       | 5 RPS indicates the application is functioning                                                              |
